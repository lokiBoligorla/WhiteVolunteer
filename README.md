# White Volunteers Foundation Website

This is a fully responsive, multi-page website for the White Volunteers Foundation NGO.

You can run this website using any of the following methods:

### Method 1: Direct File Open (Easiest)
Simply double-click the `index.html` file in the project folder. It will open in your default web browser.
*   **Path:** `white_volunteer_fundation/index.html`

### Method 2: VS Code Live Server (Recommended)
If you are using Visual Studio Code:
1.  Install the **Live Server** extension (by Ritwick Dey).
2.  Right-click on `index.html`.
3.  Select **"Open with Live Server"**.
4.  The site will automatically launch at `http://127.0.0.1:5500`.

### Method 3: Using Node.js (npx)
If you have Node.js installed, you can run a local server:
1.  Open your terminal/command prompt in the project folder.
2.  Run the command:
    ```bash
    npx serve .
    ```
3.  Open `http://localhost:3000` in your browser.
